## 🧠 About the Project

This portfolio is designed to reflect both my **technical skills** and **practical experience**. It highlights my work in machine learning, system design, and modern web development, with a strong focus on performance, UI/UX, and smooth animations.

---

## ⚙️ Tech Stack

- **Frontend:** Next.js (App Router), React, TypeScript
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Deployment:** Vercel

## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/Abhishek0ss/AbhishekSS_.git
cd AbhishekSS_
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open:

```
http://localhost:3000
```
